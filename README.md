# 📚 Student Task Manager

A simple Full-Stack Task Manager web application built using Python Flask and SQLite.

## 🚀 Features

- ➕ Add new tasks
- 📋 View all tasks
- ✏️ Edit existing tasks
- 🗑 Delete tasks
- 💾 Store tasks using SQLite database
- 🎨 Clean and simple user interface

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite

## ▶️ How to Run

1. Install Flask:
   ```
   pip install Flask
   ```
2. Run the application:
   ```
   python app.py
   ```
3. Open your browser and visit:
   ```
   http://127.0.0.1:5000
   ```
